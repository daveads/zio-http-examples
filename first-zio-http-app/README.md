# First zio-http app

To curl the routes defined, you can use the following commands:

1. Route: GET /text

```shell
curl -X GET http://localhost:8080/text
```

2. Route: GET /fruits/b

```shell
curl -X GET http://localhost:8080/fruits/b
```

3. Route: GET /json

```shell
curl -X GET http://localhost:8080/json
```
